# SSI-BCA

SSI-BCA's goal is to build a full tracking and stats network for Battle Core Arena, giving the remaining community a place to see their match history, compare stats, and keep the game going long after the official servers went dark.

---

## Projects

### [BCA-Tracker](https://github.com/SSI-BCA/BCA-Tracker)
A real-time match tracker for Battle Core Arena. Reads live game data directly from the client and tracks everything the in-game UI never fully surfaced: kills, damage, ability usage, shield and impulse stats, gravity control, and more. Matches are saved locally and organized by date. Supports Backup 3v3, FFA, and Q-Ball.

Currently in Alpha, not yet public.

### [BCA-Hub](https://github.com/SSI-BCA/BCA-Hub)
The web platform for BCA-Tracker. Player profiles, match history, and leaderboards. Work in progress.
